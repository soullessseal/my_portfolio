# Betty 周慧萱｜設計作品集網站

這是一個個人設計作品集網站，用於展示UI/UX設計與平面設計作品。
本專案整合設計流程與前端技術，透過Figma設計系統，使用Figma MCP與Codex協助生成元件、變體和頁面，並結合Sanity建立CMS後台，使作品內容可以在後台管理與更新。

網站已部署於 Vercel。
🔗 線上網站  
https://myportfolio-iota-orpin-95.vercel.app


## 專案特色

- 使用Figma建立Design System與版面設計
- 透過Figma MCP與Codex，協助將Figma轉成前端元件
- 採用Next.js + React建立網站架構
- 使用TailwindCSS進行樣式設計
- 整合Sanity作為Headless CMS，管理作品資料
- 支援RWD響應式設計（Desktop/Mobile）
- 使用Vercel進行部署
- SEO基本優化（Open Graph / sitemap / robots）
